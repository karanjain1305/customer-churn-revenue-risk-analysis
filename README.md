Customer Churn & Revenue Risk Analysis

Overview

Analyzed telecom customer churn to identify high-risk customer segments and estimate monthly revenue at risk. The project uses Python for data cleaning, SQL for analysis, and Tableau for dashboard visualization.

Tools Used

Python  
SQL / SQLite  
Tableau  
Excel / CSV  

Key Metrics

Total Customers: 7,032  
Churned Customers: 1,869  
Churn Rate: 26.58%  
Average Monthly Charges: $64.80  
Monthly Revenue at Risk: $139,130.85  

Analysis

The analysis examined churn patterns across contract type, tenure group, payment method, internet service, monthly charges, and customer demographics.

Key Insights

Month-to-month customers had the highest churn rate.  
Short-tenure customers showed the highest churn risk.  
Month-to-month contracts represented the largest monthly revenue-at-risk segment.  
Churn patterns suggest opportunities for targeted retention strategies and pricing adjustments.
